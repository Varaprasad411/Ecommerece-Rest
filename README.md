Developed a scalable E-Commerce REST API using Spring Boot and Spring Security with JWT-based authentication and role-based access control for Admin, Merchant, and Customer modules. Implemented secure login and authorization using @PreAuthorize. Admin functionalities include managing merchants and customers, blocking/unblocking users, and approving products. Merchants can perform full CRUD operations on products. Customers can register with email verification using OTP stored in Redis and sent asynchronously, view approved products, manage cart, place orders, and complete payments via Razorpay integration. Currently learning React to build a complete end-to-end application.
